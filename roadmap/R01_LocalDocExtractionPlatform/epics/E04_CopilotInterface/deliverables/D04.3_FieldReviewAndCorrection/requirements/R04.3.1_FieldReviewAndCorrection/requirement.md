## Requirement Statement
Enable users to review extracted fields, see validation warnings, and submit corrections through the UI.

## Acceptance Criteria
1. User can edit extracted field values and submit corrections.
2. Corrections are sent to the corrections API with required metadata.
3. Validation warnings and review-required flags are visible.
4. UI supports line item edits and confirmations.
5. Clear confirmation is shown after submission.
