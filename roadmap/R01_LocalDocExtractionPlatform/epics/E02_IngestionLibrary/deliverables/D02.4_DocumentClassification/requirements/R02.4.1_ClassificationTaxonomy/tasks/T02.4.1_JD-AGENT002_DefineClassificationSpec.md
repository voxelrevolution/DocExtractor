## Objective

Define classification taxonomy and accuracy targets: document types (Invoice, Contract, Report, etc.), classification method (LLM-based), and evaluation metrics.

---

## Acceptance Criteria

1. ✅ **Taxonomy Defined** – Document types (5-10 categories) with clear definitions
2. ✅ **Classification Method** – LLM prompting approach specified
3. ✅ **Accuracy Goals** – Target precision/recall documented
4. ✅ **Test Set** – 100 sample documents classified for baseline
5. ✅ **Prompt Strategy** – Zero-shot or few-shot? Which LLM?
6. ✅ **Fallback Plan** – What if classifier uncertain?

   - [ ] Manually classify 100 sample documents

4. **Design Classification Prompt** (1 hour)
   - [ ] Prompt template with taxonomy
   - [ ] Examples if few-shot
   - [ ] Fallback: what if LLM uncertain?
