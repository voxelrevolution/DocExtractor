## Objective

Validate classification prompts: test on diverse document set (300+ documents), measure accuracy, identify failure modes, document calibration results.

---

## Acceptance Criteria

1. ✅ **Accuracy Metrics** – Precision, recall, F1 per document type
2. ✅ **Confusion Matrix** – Which types are confused? Why?
3. ✅ **Test Coverage** – 300+ documents (training + holdout)
4. ✅ **Edge Cases** – Ambiguous documents (invoice-like receipts, etc.)
5. ✅ **Confidence Calibration** – Does confidence correlate with accuracy?
6. ✅ **Failure Analysis** – Top 10 misclassifications analyzed
7. ✅ **Recommendations** – Prompt tuning or next steps
8. ✅ **QC-101 Approval** – All tests passed

---

## Test Plan

**Test Dataset:**
- 50 invoices (vary: scanned, PDF, multi-page)
- 50 contracts (vary: short, long, complex)
- 50 reports (vary: business, technical)
- 50 receipts (vary: POS, email, manual)
- 50 specs (vary: software, hardware)
- 50 other (mixed ambiguous)

**Test Execution:**
1. Run each document through classifier
2. Compare to ground truth
3. Collect metrics

**Success Criteria:**
- Precision > 85% per type
- Recall > 85% per type
- Overall F1 > 85%
- No type accuracy < 75%
