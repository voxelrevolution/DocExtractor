## SECTION 2: TASK DEFINITION

### Objective (One Sentence)
Design and implement table-aware line item parsing with provenance for invoice line items.

### Description
- Detect line item tables in invoices (digital or OCR text).
- Extract row/column structure for line items.
- Attach evidence spans or bounding-box references.
- Document failure modes and routing to review.

### Acceptance Criteria
- Line items extracted with stable row ordering.
- Evidence present for each line item row.
- Table detection handles multi-line descriptions.
- Failure modes documented with fallback path.
