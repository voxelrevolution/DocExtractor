## Requirement Statement
Provide a document viewer that renders PDFs and shows extracted fields side-by-side for review.

## Acceptance Criteria
1. User can load a document and view pages reliably.
2. Extracted fields are displayed adjacent to the document.
3. Evidence pointers can be surfaced from field selections.
4. Page navigation is responsive and stable.
5. Viewer handles missing/unsupported files with clear errors.

## Dependencies
- E03 extraction output available.
